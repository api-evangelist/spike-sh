Spike's founder describes sketching a complete redesign of the interface last Christmas, leading the team to migrate from jQuery and Bootstrap to React and rebuild components from scratch. The new version features redesigned incident pages with better content hierarchy, merged activity logs and notes into unified timelines, and native mobile apps. Key enhancements include email acknowledgment support, Slack mentions in escalations, and built-in status pages, with inspiration drawn from tools like Linear and Superhuman.
